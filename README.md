## ⚡ Installatie & Gebruik
```
# Cloning the repository
git clone https://github.com/JustinLung/your-tribe-for-life-profile-card.git

# Moving to the folder
cd pyour-tribe-for-life-profile-card

# Install NPM packages
npm install

# Run local server
npm run start
```
